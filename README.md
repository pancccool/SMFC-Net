# SMFC-Net
SMFC-Net is used for brain disease diagnostic analysis based on fMRI.
# Requirement
pip3 install -r requirements.txt

# Data
In order to use your own data, you have to provide

an T by N BOLD signal matrix for every subject (N is the number of nodes,T is the time series length).

# RESEARCH PAPERS USING THIS SOFTWARE:
